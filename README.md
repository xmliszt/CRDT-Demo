# CRDT-Demo

50.012 Network Project - CRDT Demo

> This demo runs on python3

## How to Run - Counter

```:bash
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
python CRDTCounter.py
```

## How to Run - Ordered Set

```:bash
virtualenv venv
source venv/bin/activate
pip install -r requirements.txt
python uCRDTSet.py
```
